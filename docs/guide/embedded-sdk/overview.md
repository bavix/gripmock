# GripMock Embedded SDK

Welcome to the GripMock Embedded SDK documentation. This experimental feature allows you to embed GripMock directly in your Go applications for testing purposes.

::: warning
This is an experimental feature and may change significantly between versions. Use at your own risk.
:::

The SDK gives each gRPC interaction pattern its own expectation type with a fluent builder — no generic stubs, no `Commit()`, no ceremony.
